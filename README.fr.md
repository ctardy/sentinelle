# Sentinelle

> Application Android **internationale** d'audit de vie privée et de souveraineté numérique.

Sentinelle guide l'utilisateur à travers les réglages de son smartphone Android pour l'aider à reprendre le contrôle sur ses données personnelles : identifiant publicitaire, DNS souverain, assistants vocaux, diagnostics constructeur, permissions des applications, etc.

L'application est **autonome**, **gratuite**, et **ne collecte aucune donnée**. Pas de backend, pas de compte, pas de produit d'appel. Langues prévues : FR, EN, ES, DE. Les recommandations locales (DNS souverains, autorités de protection des données) sont adaptées au pays choisi par l'utilisateur, indépendamment de la langue de l'interface.

## Principe

1. L'utilisateur sélectionne son constructeur et sa version d'Android.
2. Sentinelle propose un parcours d'audit guidé pas-à-pas.
3. Chaque étape explique le risque, ouvre directement le bon écran des Réglages Android, puis laisse l'utilisateur valider.
4. Score de souveraineté en sortie.
